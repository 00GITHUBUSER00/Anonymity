这是一个测试库，我就是先创建一个文件然后初始化一下。

添加新文件到原有仓库。
1  git init                                                  初始化
2  git pull https://github.com/00GITHUBUSER00/Anonymity      获取原仓库文件
3  git add .                                                 添加更改
4  git commit -m "添加文件"                                  提交更改到本地
6  git remote -v                                             查看远程仓库状态
7  git remote add origin https://github.com/00GITHUBUSER00/Anonymity 添加远程仓库
8  git remote -v                                             查看远程仓库状态
9  git push -u origin master                                 提交更改到远程仓库
